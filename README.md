# expression-webapp

Expression is a WebApp for Sentiment Analysis. A community where you can share opinions about topics of interest on real-time. You can start sharing opinions anonymously or sign up to follow topics and get notified when the sentiment and opinion tendency about a topic changes. Expression is a proof of concept. Developed by Innova4D research group @ Universidad de las Am√©ricas, Puebla.

Expression is developed using meteor.js and a reactive design in mind.
